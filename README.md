repository.

## Built-in Commands

### Dwarf Fortress

```bash
  p-dwarf download      # Detect and download the latest version.
  p-dwarf dependencies  # Detect and install dependencies (if possible).
  p-dwarf play          # Play Dwarf Fortress.
```
